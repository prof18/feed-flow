Note April 18, 2025

A new home for FeedFlow

The FeedFlow website has been rebuilt from the ground up with a new design and this updates feed you can subscribe to.

The old FeedFlow website had been running for a while and it was showing its age. Today it’s replaced with something that actually matches the quality of the app.

What changed

The new site is built with the same Hugo static site generator, but with an entirely new custom theme. A few things worth calling out:

This updates feed. Every release and announcement will be posted here. You can subscribe to the RSS feed at /updates/index.xml — including right inside FeedFlow itself.

Better FAQ. The FAQ now covers sync setup for FreshRSS, Feedbin, Miniflux, BazQux, and more — plus answers to the most common questions.

25+ languages. The site is translated by the same community that translates the app on Weblate. If your language is missing, join the translation project.

As always, if something isn’t working or you have feedback, open an issue on GitHub.
